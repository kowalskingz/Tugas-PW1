10 Shortcut Yang Wajib di Ketahui pada Windows 10
By Awaludin Siking

Saat ini pengguna Windows 7 dan Windows 8,1 sudah bisa upgrade ke versi windows terbaru, yaitu Windows 10 secara gratis. Dalam versi windows terbarunya ini , Microsoft juga menambahkan beberapa shortcut baru seperti untuk mengakses Cortana digital assistance atau juga Virtual Desktop.




Berikut ini daftar shortcut untuk para pennguna Windows 10


Tombol Windows + Tab
Shortcut ini ada di versi windows sebelumnya, yang berfungsi untuk berpindah layar. namun microsoft membuat tampilan baru . di bagian bawah layar anda dapat menggunakan virtual desktop switcher untuk berpindah antar virtual desktops.

Tombol Windows + Tombol Arah [kiri] [kanan] [atas] [bawah]
Shortcut ini berfungsi untuk mengatur tampilan screen seperti minimize.

Tombol Windows + G
Shortcut ini akan membuka windows 10 game bar screenshot. disini anda dapat merekam game videos dari windows 10 games.

Alt + F4
Berfungsi untuk menutup windows yang sedang di gunakan, jika anda berada di desktop lalu menggunakan shortcut ini maka akan muncul kotak dialog pilihan untuk shutdown atau restart.

Tombol Windows + Shift + [Left][Right]
shortcut ini dapat digunakan saat menggunakan multiple monitored, fungsinya untuk menggeser jendela dari satu monitor ke yang lain.

Alt + Space
Shortcut ini salah satu yang patut dicoba pada windows 10, berfungsi untuk mengatur jendela yang sedang digunakan.

Windows Key + Ctrl + D
Berfungsi untuk beralih ke Virtual Desktop

Windows + Space
berfunhsi untuk berpindah bahasa inputan pada keyboard

Windows + I
Berfungsi untuk membuka setting windows 10

Windows + L
shortcut ini digunakan untuk mengunci windows 10

terakhir shortcut bonus Ctrl + alt + arah (kiri , kanan ,atas atau bawah), hehehe. sekian artikel tentang 10 Shortcut Yang Wajib di Ketahui pada Windows 10.
